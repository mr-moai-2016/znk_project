はじめてのコマンドプロンプト(Windows)

ここではコマンドプロンプトを開く方法を説明する.

これからコマンドプロンプトにて作業を行いたいフォルダをShiftキーを押しながら右クリックする.
ブルダウンメニューが表示されるので、「コマンドウィンドウをここで開く」を選択する.

Shiftキーを押しながらというのがここでのポイントである.
そうしないと「コマンドウィンドウをここで開く」という項目が出現しないのだ.

すると以下のような真っ黒なウィンドウが表示されるはずである.
これがコマンドプロンプトのウィンドウである.



WindowsXP以前やWindows10の場合、上記の方法が使えないことがある.
その場合は、スタートメニューから開く方法もある.
「スタートメニュー？なんですかそれは？」な方はいるだろうか？
画面一番左下にあるWindowsのロゴが入ったボタンがそれである.
「そんなものは見当たらない…」な方は…やむを得まい.
「コマンドプロンプト Windows8」をキーワードにしてググって欲しい.

スタートメニューを開いたら以下の手順で項目を選ぶ.
Windowsの種類によっては手順が多少異なるので、それぞれについて記述しておく.

  • WindowsXPの場合
    「プログラム」⇒「アクセサリ」⇒「コマンドプロンプト」の順となる.

  • Windows7の場合
    「すべてのプログラム」⇒「アクセサリ」⇒「コマンドプロンプト」の順となる.

  • Windows10の場合
    「すべてのアプリ」⇒「Windows システムツール」⇒「コマンドプロンプト」の順となる.

  • Windows8/8.1の場合
    スタートメニューに該当するボタンないのだが、右クリックした時に画面一番右下に現れる
    「すべてのアプリ」ボタンを押して欲しい. すると画面上に「コマンドプロンプト」のアイコンが現れる.
    どうしてもわからない場合はここであれこれ言うよりやはりググってもらった方が早いだろう.

スタートメニューからコマンドプロンプトを開いた場合、実際に作業を行うフォルダまでコマンドプロンプト上で移動する必要がある.
例えば今現在C:ドライブ内におり、その中のZenkakuというフォルダに移動したいならば、以下を実行する.

cd c:\Zenkaku

他のドライブ、例えばD:ドライブ内のero_gazoフォルダに移動したい場合には、
まず以下を実行し、現在のドライブ(カレントドライブ)を変更しなければならない.

d:

その上で cd コマンドで目的のフォルダへ移動する.

cd d:\ero_gazo

最近のコマンドプロンプトではフォルダパスを入力する時にtabキーを使えば、
自動的な補完ができるので是非活用しよう.
(tabキーを続けて連打すると補完の候補を次々と変えていくこともできる)

コマンドプロンプトを閉じるにはキーボードから exit と入力してEnterキーを押すか
普通にウィンドウ右上にある x ボタンを押す.

以上がコマンドプロンプトを開く方法だ！
全くの初心者の方は10回くらい開けて閉じてを繰り返し、確実にマスターしておくことォ！
