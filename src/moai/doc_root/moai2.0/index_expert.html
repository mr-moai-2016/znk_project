上級マニュアルのご案内

ここではMoaiとそれに付随するシステムの深部に関するハイレベルな解説記事が掲載されています.
ここにあるすべてのドキュメントはエキスパートな方向けのものであり、通常のユーザーは読む必要はありません.

上のリンクは階層構造となっており、現在あなたは「上級マニュアル」の「入り口」を閲覧されている状態です.
「上級マニュアル」には多くの記事があり、上のリンクの２段目に列挙されているものたちがそれに該当します.
ここではそれらの概要を紹介しましょう.

Moaiエンジン
この記事はMoaiエンジン本体に関して説明したリファレンスです.
主にMoaiエンジンの設定画面に関するものや、フィルタ機能、MoaiCGIおよびその認証、新防御システムXhrDMZ、
不要な接続のブロック機能(広告のカットにも応用できます)、その他の応用的な機能についてです.


CustomBoyエンジン
この記事はCustomBoyエンジンについて説明したリファレンスです.
CustomBoyとは、Moai/Easterの仮想環境を詳細にカスタマイズするエキスパートユーザ向けのツールです.
このツールの開発と各種情報の提供は高速ヤキソバ変換氏にご協力頂きました. 彼女にはこの場を借りてお礼申し上げます.


HowToコンパイル
この記事ではMoaiのソースコードをビルドする方法について解説しています.
Moai本体やシステムに付属するの全ソースコード(ライブラリも含む)をコンパイルする方法について、割とゼロベースから解説してあります.
コンパイルなんてしたことない方でも根気よく読めば出来ると思います(多分)
コンパイル環境は MinGW、VC、Cygwin、MSYS、Linux、Android NDK と比較的多くの環境を取り扱っています.


Moai CGI Developers
Moai CGI は Moaiとは独立した単独のCGIスクリプトを実行します.
従って、あなたがCGIスクリプトを独自に開発し、それをMoai CGI上で実行させることもできるのです！
ここではこのようなWebアプリケーションを独自に作ってみたい野心溢れる方向けに、その開発方法を解説します.
そして最終課題としてProxyFinderを例として挙げ、このようなツールが開発できるレベルを目指します.
