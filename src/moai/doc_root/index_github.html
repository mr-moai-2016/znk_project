はじめに

我々「全日本ネットワーク研究所(通称ZNK)」では、現在 Moaiと呼ばれるシステムを開発して皆様に提供しております.
このシステムはすべてオープンソースであり、Windows、Linux、そしてAndroid向けに開発されたものです.

What's new?

2020/03/01: Moai Ver2.2.3 リリース(Android APK版のみ). New!
マイナーバージョンアップとなります.
またこのリリースは Moai Android APK版だけの更新となりますので
Moai PC版を使われる方は関係ありません.



  • Moai PC 版を使われる方

    バージョン2.2.2が最新版となりますのでそちらをご覧ください.

    moai-v2.2.3-android.apkをダウンロードしてインストールを行ってください.

2020/01/21: Moai Ver2.2.2 リリース.
マイナーバージョンアップとなります.
  • Moai PC 版を初めてインストールされる方

    まず Moai Ver2.2 ―ResposiveStone― を新規インストールして下さい.
    詳しくはPC版のインストール方法をご参照ください.
    新規インストールが終わったら、さらにMoaiアップグレードを行って最新バージョンへと更新させます.

  • Moai PC 版を既にインストールしている方

    Moaiアップグレードにより、ただちに最新バージョンにすることができます.
    ただしVer2.2より古いバージョンをお使いの場合、これを何回か繰り返す必要があります.
    たとえばVer2.1をお使いの場合、まずVer2.2へアップグレードし、次にVer2.2.2へアップグレードするといった具合に２段階になります.

    Moaiアップグレードについての詳しい解説はこちらをご参照ください.

  • Moai Android 版を使われる方

    バージョン2.2.3が最新版となりますのでそちらをご覧ください.
    Android版に関しては、現時点では Moai アップグレードをサポートしていません.

Moaiのご紹介

これは何？
Moaiはポータブルウェブサーバシステムです. (ローカルプロキシとして利用することもできます).
詳細については、Moaiの特徴をご覧下さい.
FAQも併せてご覧頂くと、あなたの所望の機能が見つかるかもしれません.

動作環境
Moaiシステムは以下の環境であなたのブラウザと協調動作します.


Windows2000以降

Linux x86/x86_64

Android 4.0以降

ブラウザはあなたのお好きなものをご利用頂けます.

スクリーンショット
Moaiが起動中はあなたのブラウザで以下のような画面を表示させることができます.
ここからMoaiシステムが提供する様々なWebアプリケーションをブラウザ上で利用できます.
例えば以下で紹介しているEasterやVirtualUSERSなどがその代表的なものです.



ダウンロードおよびインストール
Moaiのインストールが初めての方はインストールガイドをまずご参照ください.
ダウンロード先のリンクもそちらに示してあります.
既にMoaiをインストールされている方は、Moaiのアップグレード方法を参照してください.

これは何？
VirtualUSERSはMoaiシステム上で動くウェブアプリケーションの一つです.
Easterでの仮想ブラウジングの際に使われる仮想環境を、ボタン一つで自動的に構築/再構築するツールです.

スクリーンショット
VirtualUSERSの画面に入るとあなたのブラウザで以下のような画面が表示されます.
この画面の「VirtualUSERS」ボタンを押すだけで仮想ブラウザ環境の構築が完了します.
設定等はほとんどなく、初心者でも簡単に使うことができます.
